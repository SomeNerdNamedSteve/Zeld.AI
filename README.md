<h1>Zeld.AI</h1>

<h2>About</h2>
	Zeld.AI is an Artificial Intelligence System to play the NES game 'The Legend of Zelda'

<h2>Technical Aspects</h2>
Zeld.AI was made in Python 3.5 with the following packages installed via PyPi (pip):
<ul>
	<li>opencv-python</li>
	<li>Pillow</li>
	<li>tensorflow</li>
	<li>numpy</li>
	<li>pyautogui</li>
</ul>
